## Learning Path — Tutedude DevOps

A structured curriculum covering the full DevOps lifecycle from Linux fundamentals to production-grade cloud deployments.

<br/>

**Foundation**
- Linux command line, server management, and Vagrant for virtualization
- Networking basics — client-server models, request/response, protocols
- Bash scripting — variables, conditions, loops, task automation
- Python scripting — scripting fundamentals, automation workflows

**Version Control**
- Git internals, branching strategies, collaboration workflows
